## 如何测试算法正确性？

`yarn test:算法名`可以测试相关算法，比如`yarn test:quick-sort`就可以测试快排的正确性。这里我将测试的结果重定向到了`stdout.log`跟`stderr.log`中，具体测试结果可以在这两个文件中进行查看和排查问题。
